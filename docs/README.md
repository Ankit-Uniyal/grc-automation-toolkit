# Pillar Guides Index (`docs/`)


This folder holds the 21 GRC pillar guides. Each guide is self-contained and
follows the same structure: **what to automate -> data model -> copy-paste code
-> scheduling & ownership -> maturity ladder.** Start at the top, or jump to the
pillar you need.


> New here? Read the root [README](../README.md) first, then [DIY-GUIDE](../DIY-GUIDE.md)
> and [CONVENTIONS](../CONVENTIONS.md). The 90-day rollout order lives in guide 09.
